## Tech Stack

This project demonstrates my ability to connect different layers of an application:

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend:** Java (HttpServer)
* **Storage:** File-based Database (`inventory.txt`)
* **Communication:** RESTful principles using JSON and Fetch API
* **DevOps/Deployment:** GitHub Pages (Frontend hosting)

---

## Key Features

* **Real-time Data Entry:** Add product name, price, and quantity.
* **Dynamic Deletion:** Remove items from the list with a single click.
* **Persistent Storage:** Data is stored in a text file on the server, ensuring it remains even after a server restart.
* **Live Console Logging:** The Java backend provides real-time logs in the terminal for every request received.
* **CORS Management:** Handled Cross-Origin Resource Sharing to allow secure communication between the browser and the local server.

## How to Run Locally

Since the backend runs on Java, follow these steps to see the full functionality:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/irumipramisha/inventory-management.git](https://github.com/irumipramisha/inventory-management.git)
    ```
2.  **Run the Java Server:**
    Open your terminal/IDE and run:
    ```bash
    javac InventoryServer.java
    java InventoryServer
    ```
    *The server will start on `http://localhost:8080`.*
3.  **Open the Frontend:**
    Simply open `index.html` in your browser or visit the Live Link.
